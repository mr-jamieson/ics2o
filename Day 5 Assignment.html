A simple number guessing game is played between the computer and one player. The algorithm follows:

  • Determine a secret number in a range
  • Prompt the player for a number in that range
  • Compare the player’s number to the secret number.
  • If the player’s number matches the secret number, then display a "You won!" message.
  • Otherwise display a "hint" message - this must be the result of a method call!
